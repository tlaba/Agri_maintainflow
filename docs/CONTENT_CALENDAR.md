# MaintainFlow Ag — Content Calendar (Weeks 1–6)

_Channels: **WhatsApp** (Status + broadcast/groups) and **short video** (Reels / TikTok / YT Shorts). Companion to `MARKETING_STRATEGY.md`._

## How to use this
- **Cadence:** 3 posts/week per channel (Mon / Wed / Fri). Adjust to capacity — consistency beats volume.
- **Format-first:** every post is built for a phone, sound-off, low-data. Lead with a hook in the first 2 seconds / first line.
- **Always end with one CTA** (varies): *Get it free → [link]* · *Reply to try it* · *Forward to a farmer who needs this*.
- **Reuse assets you already have:** 15s reels (9:16 / 1:1 / 16:9), 10 feature screenshots, FB banner, flyer, partner one-pager.
- **Localise:** mix English + Setswana; keep the livestock + finance hooks central.
- **UTM/track:** use distinct short links per channel where possible to learn what works.

### Theme rotation (4 content pillars)
1. 🌱🐄 **Product value** (whole-farm, livestock, offline)
2. 💰 **Money & finance literacy** (cost/ha, loan-ready)
3. 🎓 **Practical farming tips** (seasonal, crop + livestock)
4. 👨🏾‍🌾 **Farmer stories / social proof**
